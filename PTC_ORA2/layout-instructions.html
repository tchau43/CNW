Grid Layout Guide

Overview

This page provides instructions on how to use a responsive 15-column grid layout system. Follow the guidelines below to set up your layout effectively and achieve consistency across pages.

Grid System Specifications

  • Columns: 15 equal-width columns (numbered 1 to 15)
  • Rows: 8 equal-height rows (numbered 1 to 8)
  • Horizontal placement: grid-column: start/end to span columns
  • Vertical placement: grid-row: start/end to span rows

Basic Examples

Full-width section: grid-column: 1 / 16;
Left half: grid-column: 1 / 8;
Right half: grid-column: 8 / 16;
Top two rows: grid-row: 1 / 3;

Advanced Tips

  • Use grid-template-columns: repeat(15, 1fr); in your CSS to create 15 equal columns.
  • To center content, use justify-content: center; and align-content: center;.
  • For spacing, apply gap: 10px; or a similar value to add spacing between rows and columns.

Common Layout Patterns

Header (full width): grid-column: 1 / 16; grid-row: 1 / 2;
Sidebar (left third): grid-column: 1 / 6; grid-row: 2 / 8;
Main Content (right two-thirds): grid-column: 6 / 16; grid-row: 2 / 8;
Footer (full width): grid-column: 1 / 16; grid-row: 8 / 9;
